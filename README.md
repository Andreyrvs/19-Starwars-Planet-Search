## Instalando Dependências

* clone o repositorio

  ```bash
    git clone git@github.com:Andreyrvs/starwars-planet-search.git
  ```

* instale as dependências:

  ```bash
    npm install
  ```

## Executando aplicação

* Para rodar o front-end:

  ```bash
    npm start
  ```

## Executando Testes

* Para rodar os testes de linter:

  ```bash
    npm run lint
  ```
